# MGS.MonoUI

## Summary

UI management in Unity app development.

## Ability

- MonoUILoader to load UI asset.
- MonoUIManager to Create/Find/Destroy UI instance.
- MonoUI to Show/Focus/Close UI panel.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.Singleton.git?path=/Assets
  https://github.com/mogoson/MGS.AssetLoader.git?path=/Assets
  https://github.com/mogoson/MGS.Adaptive.git?path=/Assets
  https://github.com/mogoson/MGS.MonoUI.git?path=/Assets
  ```
## Samples

- Unity --> Window --> Package Manager --> Packages-Mogoson --> Mono UI --> Samples.
